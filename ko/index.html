    PDFMathTranslate

    PDF 과학 논문 번역 및 이중 언어 비교.

    • 📊 수식, 차트, 목차 및 주석 보존 (미리보기).
    • 🌐 다양한 언어 및 다양한 번역 서비스 지원.

    미리보기

    온라인 서비스 🌟

    Note

    pdf2zh 2.0 은 현재 온라인 데모를 제공하지 않습니다

    다음 데모 중 하나를 사용하여 저희 애플리케이션을 시험해 볼 수 있습니다:

    • v1.x 공개 무료 서비스 설치 없이 온라인에서 이용 가능 (권장).
    • Immersive Translate - BabelDOC 매월 1000 페이지 무료 이용 가능 (권장)

    데모의 컴퓨팅 리소스는 제한되어 있으므로 남용하지 않도록 주의해 주세요.

    설치 및 사용법

    설치

    1. Windows EXE Windows 용 추천
    2. Docker Linux 용 추천
    3. uv (a Python package manager) macOS 용 추천

    사용법

    1. WebUI 사용
    2. Zotero 플러그인 사용 (서드파티 프로그램)
    3. 명령줄 사용

    다양한 사용 사례에 따라 우리 프로그램을 사용하는 별도의 방법을 제공합니다. 더 많은 정보는 이 페이지 를 확인하세요.

    고급 옵션

    자세한 설명은 각 옵션의 전체 목록을 확인하기 위해 고급 사용법 문서를 참조하세요.

    2 차 개발 (API)

    현재 관련 문서가 제공되지 않습니다. 나중에 보충될 예정이니, 조금만 기다려 주세요.

    언어 코드

    필요한 언어로 번역하기 위해 어떤 코드를 사용해야 할지 모르겠다면 이 문서 를 확인하세요.

    감사의 말

    • Immersive Translation 은 이 프로젝트에 활발히 기여하는 기여자들을 위해 매월 Pro 멤버십 교환 코드를 후원합니다. 자세한 내용은 CONTRIBUTOR_REWARD.md 에서 확인하세요.

    • SiliconFlow 는 이 프로젝트를 위해 대규모 언어 모델 (LLM) 로 구동되는 무료 번역 서비스를 제공합니다.

    • 1.x 버전: Byaidu/PDFMathTranslate

    • 백엔드: BabelDOC

    • PDF 라이브러리: PyMuPDF

    • PDF 파싱: Pdfminer.six

    • PDF 미리보기: Gradio PDF

    • 레이아웃 파싱: DocLayout-YOLO

    • PDF 표준: PDF Explained, PDF Cheat Sheets

    • 다국어 폰트: BabelDOC-Assets 참조

    • Asynchronize

    • Rich logging with multiprocessing

    • 문서 국제화 (i18n) 는 Weblate 를 사용합니다.

    코드 제출 전에

    우리는 pdf2zh 를 더 나은 방향으로 발전시키기 위해 기여자들의 적극적인 참여를 환영합니다. 코드를 제출하기 전에 행동 강령 과 기여 가이드 를 참고해 주세요.
